# 📊 Pesquisa de Satisfação - TudoWeb

Este projeto foi desenvolvido com o objetivo de simular uma pesquisa de satisfação com clientes da empresa **TudoWeb**, utilizando a linguagem **Python**.

## 🎯 Objetivo

Coletar dados de clientes sobre o atendimento prestado e apresentar um resumo com base nas respostas.

## 🧠 Funcionalidades

* Coleta de dados dos entrevistados:

  * Nome
  * Idade
  * Opinião sobre o atendimento
* Classificação das respostas:

  * 1 → Excelente
  * 2 → Bom
  * 3 → Ruim
* Contabilização dos resultados:

  * Total de respostas **EXCELENTE**
  * Total de respostas **RUIM**

## 🔁 Estruturas utilizadas

* Estrutura de repetição `for`
* Estruturas condicionais `if`, `elif`, `else`

## 💻 Como executar o projeto

### 1. Clone o repositório

```bash
git clone https://github.com/Samarasoares1106/pesquisa-de-satisfacao
```

### 2. Acesse a pasta do projeto

```bash
cd pesquisa.py
```

### 3. Execute o programa

```bash
python pesquisa.py
```

## 🖥️ Terminal: o que é e como usar

O **terminal** é uma ferramenta onde você digita comandos para executar ações no seu computador.

Você pode utilizar:

* Prompt de Comando (CMD)
* PowerShell
* Terminal do VS Code (recomendado)

### ✔️ Usando o terminal no VS Code:

1. Abra a pasta do projeto
2. Clique em **Terminal → Novo Terminal**
3. Digite os comandos normalmente
